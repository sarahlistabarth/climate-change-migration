# Sea level data 

This repo downloads sea level data for a university project.

The [Sea Surface Height](https://confluence.ecmwf.int/display/CKB/Sea+Level+version+DT2024%3A+Product+User+Guide+and+Specification) for 2000--2025 is downloaded from Copernicus, from the series [ORAS5 global ocean reanalysis monthly time-series data from 1958 to present](https://cds.climate.copernicus.eu/datasets/reanalysis-oras5-timeseries).
I aggregate them at the grid level at the yearly level, using the median.
